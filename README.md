# Ensemble-framework-for-epidemic-surveillance

This repository contains data, simulation code, and algorithms, including comparison methods, for the research article:

"Monitoring the time-varying process of infectious disease using an ensemble framework of modified SEIR model and machine learning-based control chart."

Overview

— The repository provides:

— Epidemiological data used for modeling and validation.

— Simulation scripts for generating time-varying epidemic scenarios.

— Implementation of the proposed ensemble framework, integrating a modified SEIR model with machine learning-based control charts.

— Code for comparison algorithms to evaluate performance against baseline monitoring approaches.
